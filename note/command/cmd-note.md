## 常用命令行合集


- dir 			:显示当前目录下的所有文件和文件夹
- mkdir <fielname>			:新建文件夹
- md <fielname>				:新建文件夹
- rd /s 	<path>			:删除文件夹以及文件夹下所有的子文件夹
- del <path>				:删除文件，或者文件夹下面的文件
